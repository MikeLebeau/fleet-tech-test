# Fleet Tech Test

## Install
It will work with your favorite package manager (npm, pnpm, yarn etc)

### With pnpm 
> pnpm i

## Setup

Make a copy of the .env file
> cp .env .env.local

Replace the 'something' by your Token and you're good to go!

## Run
> npm dev
